経歴

倉﨑敦大 （くらさき あつひろ）1995年2月24生まれ

大学在学中、3DCG制作を行っている株式会社Studio GXへ入社。遊技機演出やスマホゲームのキャラクターやアイテム制作案件で制作進行を担当。その後同社退職後、フリーランスとして3DCG制作案件を請け負う。2014年9月、ベトナム現地のオフショア開発会社と業務提携を行い、株式会社Creation Linksを創業。オフショアでのweb・システム開発受託事業を4名でスタート。その後、創業2年目でキャッシュフロー悪化の為、同社での事業活動を休止。2016年11月に知人の紹介でカンボジアの現地オフショア開発会社であるNanita Tech, CO., LTD.にブリッジエンジニア兼デザイナーとして入社、クライアントと折衝・エンジニアのアサインからスケジュール・ソースコードの管理を行う。その後、同社を退職し、再びフリーランスとして2017年11月〜2018年4月までベトナム法人EVOLABLE ASIA, CO., LTD.のWebディレクター兼デザイナーとして参画、同社のWeb制作案件のデザイン制作・ディレクションを行う。同年5月から約4ヶ月間インドを放浪し、9月に日本へ帰国、同年11月よりブリティッシュアメリカンタバコジャパン合同会社で業務委託として同社デジタルマーケティング部署でluckystrike.jp / Kent.jpのマーケティング戦略からエージェンシー・ベンダーの管理を行う。現在は国内の複数企業でweb開発案件の制作ディレクションからワイヤーフレーム・デザイン制作を行なっています。
